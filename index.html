Legacy project modernized

Cerro Castor

A refreshed static presentation layer for the legacy Cerro Castor project. The outdated build stack is left in the archive, while the public entrypoint is now simple, fast, and deployable.

StaticResponsiveVercel ready

Project Notes

This project originally shipped with a more complex build chain and many generated assets. The modern version keeps the character but simplifies the delivery path.
The palette and imagery are intentionally close to the source project so the Vercel link feels like the real site, not a placeholder.
The page is now static, responsive, and ready to be extended later if the original content is rebuilt in a modern framework.
